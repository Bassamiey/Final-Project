About Simple Task Management System

A simple, clean task manager built for our class project to help students stay organized.

What is this project?

Our Project Task Management System Board is a platform we built to help our team stay organized and keep track of all our assignments in one place. We created this tool to make collaboration easier, allowing us to monitor our progress on different tasks and ensure we hit our deadlines without the usual group project confusion.
Who built this?

The website is built by students of BSIT 2nd year Block 3 for their final project in Integrative Programming and Web System Technologies.
How it works

Our board works by breaking down the whole project into smaller, manageable tasks that move through different stages. First, we list everything that needs to be done in the To Do column. As we start working, we move those tasks to In Progress, and finally to Done once they are finished. This way, everyone in the team can see exactly what’s being worked on, who is responsible for what, and how much we have left to do before the deadline.
Why use it?

We built this system to keep our group work organized and make sure we never miss a deadline. By moving tasks from "To Do" to "Done," we can easily see how much progress we’ve made and who is working on what, which helps us avoid the stress of messy group chats and last-minute rushing.
