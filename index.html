About Project Task Management Board

A simple, clean task manager built for our class project to help students stay organized.

What is this project?

The Project Task Management Board is a lightweight Kanban-style tool we built to manage school assignments and group work. It runs right in your browser so you can keep track of what needs to get done without any complicated setup.
Who built this?

We built this as a learning project to practice our web development skills. We stuck to the basics—using just HTML, CSS, and JavaScript. No fancy frameworks or external libraries, just clean code and a lot of trial and error!
How it works

You can create tasks, move them between columns as you work on them, and delete them when you're finished. Everything saves automatically in your browser's storage, so your progress won't disappear when you refresh the page.
Why use it?

Sometimes big project management tools are just too much for a simple school assignment. We wanted to make something fast and easy to use that gets out of your way so you can actually focus on finishing your homework.
